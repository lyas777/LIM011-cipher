KRIPTTA

Mensajería Secreta

Kriptta es una aplicación web la cual puede ser usada como "Herramienta de mensajería secreta", esta aplicación ofrece el servicio de encriptar mensajes en línea. Las conversiones se realizan mediante el Navegador Web sin interactuar con algun servidor, lo que significa que la información que se ingresa no se almacena en ningun servidor ni base de datos, por lo tanto esta aplicación es segura y confiable.
